> RoLLodeQc utility to fetch a single GitHub user.

## Install
```
$ npm install --save rollodeqc-gh-user
```

## Usage
```js
const rollodeqcGhUser = require('rollodeqc-gh-user');

rollodeqcGhUser('unicorns');
//=> 'unicorns & rainbows'
```
